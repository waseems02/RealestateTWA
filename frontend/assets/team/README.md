# Team photos

Drop your headshot here as **JPG**, exactly matching one of these filenames:

- `waseem.jpg`
- `tareq.jpg`
- `abed.jpg`

Recommended: square, 400×400 or larger, decent lighting. The site clips it
into a circle automatically and shows a coloured-initial fallback until the
file exists — so pushing the image is enough to make it appear.
